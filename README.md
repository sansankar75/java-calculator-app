# Java Calculator Application

A simple and intuitive calculator application built with Java Swing for Windows.

## Features

- Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
- Decimal point support
- Delete last digit functionality
- Clear display
- Negative number toggle
- Clean and user-friendly GUI with green-themed panel

## Requirements

- Java Development Kit (JDK) 8 or higher
- Windows Operating System

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/sansankar75/java-calculator-app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd java-calculator-app
   ```

3. Compile the Java file:
   ```bash
   javac calculator.java
   ```

4. Run the application:
   ```bash
   java calculator
   ```

## Usage

- **Number Buttons (0-9)**: Click to enter numbers
- **Operator Buttons (+, -, *, /)**: Perform arithmetic operations
- **Decimal (.)**: Add decimal point to numbers
- **Equal (=)**: Calculate and display result
- **Delete**: Remove the last entered digit
- **Clear**: Clear the entire display
- **(-)**: Toggle between positive and negative numbers

## Technical Details

- Built using Java Swing for GUI components
- Implements ActionListener for button event handling
- Uses GridLayout for organized button placement
- Window size: 420x550 pixels
- Custom font: Ink Free, Bold, 30pt
